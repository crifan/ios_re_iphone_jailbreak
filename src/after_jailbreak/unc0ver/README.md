# unc0ver越狱后

用unc0ver给iPhone越狱后，重启后，即可完成越狱。

## 【首次】初始化Substitute

iPhone重启后，往往会看到：

锁屏界面显示：

* `Substitute  - request for permission`
  * ![rebooted_jailbroken_substitute](../../assets/img/rebooted_jailbroken_substitute.png)
  * ![iphone7_rebooted_notice_substitute](../../assets/img/iphone7_rebooted_notice_substitute.jpg)

解锁后，点击：Always Allow

* ![substitute_request_for_permission](../../assets/img/substitute_request_for_permission.png)

注：有时候是，此弹框，会出现unc0ver越狱期间：

* ![unc0ver_jailbreaking_substitute_popup](../../assets/img/unc0ver_jailbreaking_substitute_popup.png)

然后点击`Substitute`进入界面，查看到相关设置：

* ![unc0ver_jailbroken_substitute_1](../../assets/img/unc0ver_jailbroken_substitute_1.png)
* ![unc0ver_jailbroken_substitute_2](../../assets/img/unc0ver_jailbroken_substitute_2.png)

## unc0ver成功越狱后的效果

* 用unc0ver越狱成功后
  * 越狱成功后的桌面：有了`unc0ver`和`Substitute`
    * ![unc0ver_jailbroken_iphone7_desktop](../../assets/img/unc0ver_jailbroken_iphone7_desktop.png)
  * 打开unc0ver后会显示：已越狱
    * 旧版unc0ver显示：`Re-Jailbreak` 
      * ![unc0ver_re_jailbreak](../../assets/img/unc0ver_re_jailbreak.png)
    * 新版unc0ver显示：`Jailbroken`
      * iOS 14.3的iPhone8，用unc0ver 8.0.2越狱后
        * ![unc0ver_802_jailbroken](../../assets/img/unc0ver_802_jailbroken.png)
  * 爱思助手检测到iPhone是：已越狱
    * iPhone8
      * ![i4tools_show_jailbroken_iphone8](../../assets/img/i4tools_show_jailbroken_iphone8.png)
  * 其他越狱工具能正常打开
    * 对比：如果越狱失败，则打开对应工具会闪退崩溃
    * Cydia
      * ![unc0ver_jailbroken_cydia_1](../../assets/img/unc0ver_jailbroken_cydia_1.png)
      * ![unc0ver_jailbroken_cydia_2](../../assets/img/unc0ver_jailbroken_cydia_2.png)
        * `iPhone9,1`
        * `iOS 13.6.1`
        * `Cydia64_1.1.36(zh-cn)`
