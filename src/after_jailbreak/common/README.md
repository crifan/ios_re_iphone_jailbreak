## 包管理器

概述：越狱后，往往会通过包管理器去安装越狱开发所需的各种插件和工具。比如OpenSSH用于ssh登录、Filza用于管理文件、TrollStore用于安装ipa等等。

具体详见独立教程：

[iOS逆向开发：越狱包管理器](https://book.crifan.org/books/ios_re_package_manager/website/)
